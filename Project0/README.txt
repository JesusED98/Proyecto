Para este proyecto el tema que decidi fue acerca de bebidas alcoholicas preparadas pero en esencial 4 porque son las que mas me han gustado, se maneja una pagina de inicio (index) donde se abarca un poco de historia y origenes de las bebidas.

Tambien se cuenta con cuatro ventanas aparte, una para cada bebida en donde se explica un poco acerca de ella, los ingredientes, la elaboracion e informacion adicional este dependiendo de la bebida.

Para realizar este proyecto se utilizo html, bootstrap, hipervinculos, listas, tablas, imagenes, archivos de estilo: CSS, SCSS y media query para cuidar los detalles de las vistas. Mas que nada utilice el material visto en clase y unas cosas que investigue sobre las librerias de bootstrap para el diseño.
